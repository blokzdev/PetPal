# Socialization (the closing window)

The critical socialization window is roughly 3–14 weeks. After ~14
weeks, novel stimuli increasingly elicit fear rather than curiosity.
Underuse this window and the dog spends a lifetime managing fears
that wouldn't have formed otherwise.

What the user should be doing during the window:

- **Variety, not intensity.** Different surfaces (gravel, grass,
  metal grates, slick floors), different sounds (vacuum, doorbell,
  traffic, kids), different humans (hats, beards, mobility aids,
  varied skin tones). Brief, positive exposure beats prolonged
  exposure.
- **Safe dog contact only.** Pre-vaccinated puppies should meet
  vaccinated, known-safe adult dogs in controlled settings — not
  random dog-park dogs. Puppy classes from reputable trainers
  vaccinate-screen attendees.
- **Owner-administered handling.** Touch paws, ears, mouth daily so
  vet visits aren't the only time the puppy is restrained.

If the user is past the window: socialization shifts from
"introduction" to "counter-conditioning" — slower, requires a
behaviorist for fearful dogs. Don't pretend the window is still open.
