# Litter box setup for a new cat

Litter box problems are almost always setup problems, not behavior
problems. Run through the basics before assuming anxiety or training.

**The N + 1 rule.** Number of litter boxes = number of cats + 1. Even
for a single cat, two boxes is the right answer. Place them in
different rooms / floors.

**Box specs the cat actually wants:**
- **Big.** Adult cat should be able to turn around comfortably; many
  store boxes are too small. A storage tote with one wall cut down
  works.
- **Uncovered, ideally.** Covered boxes trap odor for the cat (which
  is why we like them) and many cats avoid them. Try uncovered first.
- **Unscented clumping clay litter** is what most cats prefer. Scented
  litter is for humans; the cat hates it.
- **Scoop daily, full change weekly.** Cats notice 24-hour buildup.

**Placement:**
- Quiet, not next to the washing machine, not next to food/water.
- Two exits if possible — a cat that feels trapped while in the box
  is a cat that finds another bathroom.

If the user reports going outside the box, log every incident as a
`behavior/` entry: location, time, after-event behavior. Pattern
matters more than any single event. Also worth a vet check — UTI and
crystals present as litter-box avoidance, especially in male cats.
