# New cat settling

Cats acclimate to new homes on cat time, not human time. Calibrate
the user's expectations downward.

- **Single safe room first.** Bathroom or spare bedroom for the first
  3–7 days. Litter box, food, water, hiding spot. Not the whole house
  immediately — that's overwhelming.
- **3-3-3 rule.** 3 days hiding + decompressing; 3 weeks settling into
  routine; 3 months feeling fully home. The user reporting "still
  hiding after a week" is on schedule, not a problem.
- **Don't force interaction.** Sit in the room, read a book, let the
  cat approach. Eye blinks, soft conversation, treats placed nearby
  rather than offered hand-to-hand.
- **Slow introductions** to other pets and rooms — scent swap (rub a
  cloth on each animal, swap), then visual through a cracked door,
  then supervised together. Days, not minutes.

If the user reports not eating for >24 hours, that's vet territory
(hepatic lipidosis develops fast in cats). Treat as escalation.
