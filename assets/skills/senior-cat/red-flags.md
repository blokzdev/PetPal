# Senior-cat red flags

Several patterns disproportionately affect older cats. None of these
are diagnoses — they're prompts to call the vet sooner rather than
later.

**Drinking and peeing more.** Polyuria/polydipsia in an older cat
is a vet visit, not a wait-and-see. The leading causes (kidney
disease, hyperthyroidism, diabetes) all benefit hugely from early
intervention. Ask the user to track water bowl refills + litter
clumping for a few days; concrete numbers help the vet.

**Weight loss with normal or increased appetite.** Classic
hyperthyroidism. A 12+ year old cat losing weight while eating
ravenously and acting "wired" needs a T4 blood test. Weight loss
*without* appetite is a different short-list (kidney, dental, GI,
neoplasia) — same urgency, different work-up.

**Stopped jumping. Hesitates before stairs. Missing the litter box
edge.** These look like "slowing down with age" and are almost
always feline osteoarthritis, which is grossly underdiagnosed
because cats don't limp the way dogs do. A vet visit can confirm
and offer real options (gabapentin, monoclonal antibody injection,
environmental changes — see environment.md).

**Yowling at night.** Cognitive dysfunction syndrome (the cat
equivalent of dementia) shows up as nighttime vocalization,
disorientation, sleep-wake reversal, decreased social interaction.
But the same symptom list also fits hyperthyroidism, hypertension,
and pain. Vet first; assume CDS only after ruling those out.

**Bad breath, drooling, dropping food.** Dental disease is endemic
in seniors and causes ongoing low-grade pain. A dental under
anesthesia is a meaningful surgical decision in an old cat — but
untreated disease shortens life and degrades quality every day. The
risk math usually favors the cleaning.

If the user describes any of the above, the right framing is **call
your vet this week**, not "watch it" and not "this is an emergency."
PetPal can help write down what's been happening so it's ready when
they call.
